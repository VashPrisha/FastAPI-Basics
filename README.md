# FastAPI-Basics

This repository contains my step-by-step learning progress of FastAPI.

Currently implemented:
- Basic FastAPI setup
- GET endpoints
- POST endpoints
- PUT endpoints
- DELETE endpoints
- Path parameters

## How to run

1. Install dependencies  
   pip install -r requirements.txt

2. Run the server  
   uvicorn main:app --reload

3. Open in browser  
   http://127.0.0.1:8000/docs
